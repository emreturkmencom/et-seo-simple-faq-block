A lightweight, SEO-friendly Gutenberg FAQ block using HTML5 details/summary and auto JSON-LD schema.

== Description ==

**SEO Simple FAQ Block** is a native Gutenberg block designed for performance and SEO. It creates a clean, accessible FAQ section on your WordPress site without the need for heavy JavaScript libraries.

Key features:
* **Native Gutenberg Experience:** Built with API v3 standards.
* **Lightweight:** Uses HTML5 `<details>` and `<summary>` for accordion functionality (No JS on frontend).
* **SEO Optimized:** Automatically generates `FAQPage` JSON-LD schema markup.
* **Accordion Mode:** Optional mode to ensure only one item is open at a time.
* **Customizable Styles:** Choose from Default, Boxed, Bordered, or Dark Modern themes.
* **Multilingual:** Translation ready for Turkish, Spanish, German, French, and more.

== Installation ==

1. Upload the `et-seo-simple-faq-block` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the "SEO Simple FAQ" block in the Gutenberg editor.

== Frequently Asked Questions ==

= Does it require JavaScript for the accordion? =
No, it uses native HTML5 tags for a lightning-fast frontend experience.

= Does it support Schema.org? =
Yes, it automatically adds JSON-LD FAQPage schema for search engines.
